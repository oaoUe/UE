# 新制多益 TOEIC 聽力刷題

6 回模擬試題線上刷題，共 600 題，即時批改並顯示中英對照解析。

## 功能

- **6 回完整題庫**：TEST 1–6，各含 Part 1–4，共 100 題／回
- **即時批改**：作答後立即顯示正解與中英對照解析
- **作答紀錄保存**：刷新頁面後答案仍在，可繼續未完成的題目
- **倒數計時**：依照真正多益時間分配（Part 1: 5 分、Part 2: 8 分、Part 3: 17 分、Part 4: 14 分），時間到跳出提示，仍可繼續作答
- **完整作答成績**：一次做完同一回全部 4 個 Part，自動顯示總分與總時間
- **計時暫停**：切換分頁或縮小視窗時自動暫停，回來後繼續
- **跨設備同步**：透過 GitHub API 同步作答紀錄（需設定 PAT）
- **RWD**：支援手機與平板

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何環境。

```
開啟 index.html → 選擇回次與 Part → 開始作答
```

## 跨設備同步設定

1. 前往 GitHub → Settings → Developer settings → Personal access tokens → **Generate new token**
2. 勾選 `repo` 權限後產生 token
3. 開啟 `index.html` → 「☁ GitHub 同步」區塊 → 貼上 token → 按「儲存」
4. 之後每次開啟 `index.html` 會自動從 GitHub 拉取最新紀錄
5. 每個 Part 完成後自動推送紀錄到 GitHub

## 計時設定

`index.html` 下方「⏱ 計時設定」可自訂每個 Part 的作答時間（分鐘），設為 0 可關閉倒數。
